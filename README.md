# Modelo Reseñas Películas
_Un modelo IA que detecta si una reseña de una película es positiva o negativa.
Usando Machine Learning + Docker + Flask + Render_

#### Autor: Enmanuel Méndez Guzmán

Este proyecto implementa un sistema completo de análisis de sentimiento capaz de clasificar reseñas de películas en inglés como positivas o negativas.
<img width="1191" height="513" alt="image" src="https://github.com/user-attachments/assets/4de7b26e-ad0d-4492-b263-34580ac86da3" />

# Incluye:
- Modelo de Machine Learning en Python entrenado con NLTK y Scikit-Learn
- API REST construida con Flask
- Interfaz web sencilla en HTML/JS
- Contenerización con Docker

Para probar ingrese a: https://modeloiapeliculas.onrender.com/

Nota: La aplicación toma unos minutos para cargar
